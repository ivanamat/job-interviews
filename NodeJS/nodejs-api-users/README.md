# NodeJS API REST
This is an example of REST API developed with Swagger using MySQL as persistent data storage. This API manage users. You can Create, Read, Update and Delete users.

## Installation

### NodeJS
Just execute
```
npm install
```

### MySQL
Import the nodejs.sql file into your database and configure *service/databaseConnection.js*

## Run
Ececute
```
node .
```
and go to [http://localhost:8080](http://localhost:8080)
